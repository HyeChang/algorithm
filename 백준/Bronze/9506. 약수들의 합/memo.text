- 입력 받은 수 약수 구하기(오름차순 나열)
>> 약수 구하는 법
  1. 입력 받은 수의 제곱근만큼 나누어 =0 값을 구한다
   2. 구한 값들로 입력 받은 수를 각각 나누어 준다
   3. 중복 제거 + 오름차순 정렬
       ㄴ 이때 시간 복잡도는 O(√N) - 10억의 입력이 들어온다해도, 약 3만번의 연산으로 약수를 구할 수 있다.

- 해당 약수(입력 수 제외) 더하기
- (입력 받은 수) == (약수 더한 값)   ------ 비교
- 입력 마지막 -> '-1'

- 완전수 라면 : 6 = 1 + 2 + 3 (오름차순 나열)
- 아니라면  : n is NOT perfect.  출력
